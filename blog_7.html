Welcome to the Frontend Developer Career Path!

The place most people start. I did not, but how was it?

Module 1: Welcome

Not so much to learn here, but you get a introduction on what the rest of the course will be. Stipulated to 25 minutes. You do learn some coding, though.

Module 2: Web Dev Basics

99 lessons, 5hr, 14 minute. You learn how you use the developer mode in the Google browser, structure on your files, build a Google Clone, making a business card, and much of the basics in CSS coding. Quite fun, and YES - I learned alot. So much that you think you have done correct in the past, but clearly an eyeopener. Structuring files for an example, so much easier to read your own code when doing it :) The Solo-project is to make a Hometown homepage
